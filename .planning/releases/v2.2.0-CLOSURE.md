# Fechamento de Release v2.2.0 — TaxonomySystem

Este documento registra a higiene final e o fechamento global da release **v2.2.0** do Olfactory Taxonomy System, após a conclusão com sucesso da **Phase 23**.

## Resumo do Status da Release
- **Status da Phase 23:** Complete / Closed (Concluída e Fechada)
- **Promoção de Descriptor:** `lemon_peel` promovido com sucesso de candidato do corpus a seed descriptor (curado).
- **Publicação de Artefatos:** Artefatos oficiais compilados em `data/compiled/v2/` publicados após validação de invariantes.
- **Validação de Invariantes:** 7 de 7 invariantes apresentaram status **PASS**.
- **Tratamento de Conflitos (Review Queue):**
  - O conflito com `peel` (frequência=70) foi aceito como o aviso esperado **WARN-PAT03** (novo conflito pós-add de lemon_peel).
  - O candidato `lemongrass` foi preservado no `review_queue` sob o âncora `lemon`.
- **Ações de Curation Diferadas:**
  - Nenhuma ação foi tomada para `ylang_ylang` (sem alterações ou promoção).
  - Nenhuma outra curadoria foi realizada além de `lemon_peel` nesta release.
- **Política de Graphify:**
  - Nenhum commit foi realizado para arquivos sob `graphify-out/*` (sujeitos à política de não-mutação deliberada no histórico de commits do código principal).

## Documentação do Script de Validação

O script de validação automatizada foi comitado no repositório.

### Uso do Script
Para rodar a validação automatizada dos invariantes da Phase 23:
```bash
python3 scripts/validate_phase23.py
```

O script verifica a integridade de `data/compiled/v2/taxonomy.json` e `data/compiled/v2/similarity_matrix.json`, gerando um relatório detalhado de conformidade para cada um dos 7 invariantes.
