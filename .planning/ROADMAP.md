## Phase Details

### Phase 40: Low-Support Curation Planning
**Goal:** Inventory, prioritize, and select a bounded batch of low_support candidates.
**Requirements:** TRI-01, TRI-02, TRI-03
**Success criteria:**
1. Complete list of 275 candidates is inventoried.
2. A bounded batch of 25–50 candidates is selected based on evidence priority, semantic clarity, and expected curation value.
3. Selection rationale and deferral reasons are recorded.

### Phase 41: Low-Support Batch Decision Matrix
**Goal:** Produce a formal decision matrix for the selected candidate batch.
**Requirements:** CUR-01
**Success criteria:**
1. Every selected candidate has an explicit disposition in the matrix.
2. The matrix is approved before execution begins.

### Phase 42: Low-Support Microcuration Execution
**Goal:** Safely apply approved matrix decisions to the taxonomy.
**Requirements:** CUR-02
**Success criteria:**
1. Only approved decisions (seed additions, aliases, rejects, defers) are applied.
2. No implicit or unapproved mutations occur.

### Phase 43: Taxonomy v2.7 Artifact Publication
**Goal:** Validate artifacts, update metrics, and produce the closure report.
**Requirements:** ART-01, ART-02, ART-03
**Success criteria:**
1. Taxonomy invariants pass validation.
2. v2.7 artifacts are compiled and published.
3. A formal rebaseline/closure report details starting state, batch size, decisions, and final metrics.
