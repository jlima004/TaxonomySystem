<objective>
Implementar o carregador do corpus `enriched_materials.json` (~67MB, 33.742 materiais). O loader lê o arquivo completo via `JSON.parse`, aplica filtragem seletiva de campos (apenas `identity` + `olfactory` + campos estruturais), e retorna `CorpusMaterial[]` tipado. Interface assíncrona para future-proofing.
</objective>

<tasks>

### Task 1: Implementar corpus_loader.ts com filtragem seletiva

<read_first>
- src/types/corpus.ts
- src/types/index.ts
- .planning/phases/02-data-loaders/02-RESEARCH.md §2, §4
- .planning/phases/02-data-loaders/02-CONTEXT.md §Decisions D-01 a D-09
</read_first>

<action>
Criar `src/loader/corpus_loader.ts` com:

1. Função principal `load_corpus(path: string): Promise<readonly CorpusMaterial[]>`

Implementação:
- `await readFile(path, 'utf8')` com try/catch para path inexistente
- `JSON.parse(content)` com try/catch para JSON inválido
- Validar que resultado é array: `if (!Array.isArray(parsed))` → throw
- `parsed.map(map_to_corpus_material)` — filtragem seletiva

2. Função interna `map_to_corpus_material(raw: unknown): CorpusMaterial`

Mapeia APENAS os campos do type `CorpusMaterial`:
- `id`: string (copiar direto)
- `identity`: { name, canonical_name, aliases, identifiers: { cas?, einecs? } }
- `classification`: { category, category_path }
- `olfactory`: { descriptors, primary_type, odor_description, descriptor_sources } — EXCLUIR `flavor_description`
- `usage`: { found_in_nature, uses, blenders }
- `molecular`: { cid?, smiles?, molecular_weight?, xlogp?, tpsa?, rotatable_bonds? }

Campos excluídos (D-09): `text`, `physchem`, `solubility`, `safety`, `meta`

3. Função interna `map_olfactory_profile(raw: unknown): OlfactoryProfile`
   - Extrair apenas os 4 campos definidos no type: `descriptors`, `primary_type`, `odor_description`, `descriptor_sources`
   - Garantir arrays defaults vazios `[]` se campos ausentes
   - Garantir strings default `''` se campos string ausentes

Import: `import { readFile } from 'node:fs/promises'`
Import type: `import type { CorpusMaterial, OlfactoryProfile } from '../types/index.ts'`
</action>

<acceptance_criteria>
- `src/loader/corpus_loader.ts` exporta `load_corpus`
- Assinatura: `(path: string) => Promise<readonly CorpusMaterial[]>`
- Output NÃO contém campos `text`, `physchem`, `solubility`, `safety`, `meta`
- Output NÃO contém `olfactory.flavor_description`
- Cada item no output satisfaz o type `CorpusMaterial` (tsc compila)
- Campos olfativos ausentes recebem defaults (arrays vazios, strings vazias)
- Path inexistente: rejeita com mensagem mencionando o path
- JSON malformado: rejeita com mensagem de parse error
- Input não-array: rejeita com mensagem "Expected JSON array"
</acceptance_criteria>
