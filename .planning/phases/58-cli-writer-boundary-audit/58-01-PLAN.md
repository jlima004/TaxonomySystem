## Task 1: Implement `write_graph.ts` ‚Äî Graph Writer Module

<read_first>
- `src/compiler/write_outputs.ts` (atomic write pattern: `writeJsonDeterministic`, tmp‚Üírename, `cleanupTemps`)
- `src/graph_read_model/contract.ts` (`GRAPH_OUTPUT_POLICY`, `GRAPH_OUTPUT_POLICY.forbidden_output_prefixes`, `GRAPH_OUTPUT_POLICY.sanctioned_output_path`)
- `src/graph_read_model/types.ts` (`OlfactoryGraph` shape)
</read_first>

<action>
Create `src/graph_read_model/write_graph.ts` with:

1. `GraphWriteError` class extending `Error` ‚Äî includes `code: string` field for structured error reporting (`'forbidden_prefix' | 'validation_failed' | 'write_failed'`).

2. `validateOutputPath(outputDir: string): void` ‚Äî checks `outputDir` against each entry in `GRAPH_OUTPUT_POLICY.forbidden_output_prefixes`. Normalizes both sides (strip trailing `/`, use forward slashes). Throws `GraphWriteError` with code `'forbidden_prefix'` if any prefix matches. Also rejects paths containing `graphify-out` anywhere (GVAL-04 write guard).

3. `writeGraphOutput(graph: OlfactoryGraph, outputDir: string): Promise<string>` ‚Äî orchestrates:
   - Call `validateOutputPath(outputDir)`
   - `mkdir(outputDir, { recursive: true })`
   - Compose `tmpPath = join(outputDir, '.graph.json.tmp')` and `finalPath = join(outputDir, 'graph.json')`
   - Call `writeJsonDeterministic(tmpPath, graph)` (import from `../../compiler/write_outputs.js`)
   - `rename(tmpPath, finalPath)`
   - On error: `rm(tmpPath, { force: true })` cleanup, then re-throw
   - Return `finalPath`

Export `GraphWriteError`, `validateOutputPath`, `writeGraphOutput`.
</action>

<acceptance_criteria>
- `src/graph_read_model/write_graph.ts` exports `GraphWriteError`, `validateOutputPath`, `writeGraphOutput`
- `validateOutputPath` throws for paths starting with `data/compiled/`, `data/taxonomy/`, `data/inference/`, `graphify-out/`
- `validateOutputPath` also rejects any path containing `graphify-out` anywhere in the path
- `validateOutputPath` does NOT throw for `data/read-models/olfactory-graph/v2.11/`
- `writeGraphOutput` creates `.graph.json.tmp` then renames to `graph.json`
- `writeGraphOutput` creates output directory if it doesn't exist
- `writeGraphOutput` uses `writeJsonDeterministic` from `write_outputs.ts` (not a custom JSON.stringify call)
- No external dependencies added to `package.json`
</acceptance_criteria>

## Task 2: Implement `boundary_audit.ts` ‚Äî SHA-256 Boundary Audit Module

<read_first>
- `src/graph_read_model/contract.ts` (`GRAPH_OUTPUT_POLICY`, `GRAPH_ALLOWED_PRODUCTION_INPUTS`)
- `src/cli/alias_integrity.ts` (`resolveReadablePath` pattern for path resolution)
- `src/graph_read_model/write_graph.ts` (Task 1 output ‚Äî for graphify isolation guard pattern)
</read_first>

<action>
Create `src/graph_read_model/boundary_audit.ts` with:

1. Type definitions:
   ```
   BoundaryFileDigest = { path: string; sha256_before: string; sha256_after: string; unchanged: boolean }
   BoundaryAuditResult = { ok: boolean; protected_files: BoundaryFileDigest[]; graphify_out_accesses: 0; output_written: string; forbidden_prefix_rejections: string[] }
   ```

2. `hashFile(filePath: string): Promise<string>` ‚Äî reads file content with `readFile(filePath)`, returns `createHash('sha256').update(content).digest('hex')`. Uses `node:crypto`.

3. `GVAL_03_PROTECTED_FILES` constant ‚Äî the exact named files from D-13:
   - `data/taxonomy/taxonomy-seed.v2.json`
   - `data/taxonomy/descriptor_aliases.seed.json`
   - `data/taxonomy/alias_target_exceptions.v1.json`

4. `discoverProtectedFiles(baseDir: string): Promise<string[]>` ‚Äî combines `GVAL_03_PROTECTED_FILES` with dynamically discovered files under `data/compiled/v2/` (using `readdir`). Resolves all paths relative to `baseDir`. Returns sorted array of absolute paths.

5. `capturePreDigests(files: readonly string[]): Promise<Map<string, string>>` ‚Äî hashes all files, returns map of `path ‚Üí sha256`.

6. `comparDigests(preDigests: Map<string, string>, files: readonly string[]): Promise<BoundaryFileDigest[]>` ‚Äî re-hashes files, compares with pre-digests, returns array of `BoundaryFileDigest`.

7. `runBoundaryAudit(outputWritten: string, baseDir: string, preDigests: Map<string, string>): Promise<BoundaryAuditResult>` ‚Äî re-hashes, builds result with `graphify_out_accesses: 0` (static ‚Äî GVAL-04), computes `ok` from all `unchanged === true`.

Export all types and functions.
</action>

- `GVAL_03_PROTECTED_FILES` contains exactly 3 named files from D-13
- `hashFile` uses `node:crypto` `createHash('sha256')`
- `discoverProtectedFiles` includes both named files and `data/compiled/v2/*` discovered via `readdir`
- `runBoundaryAudit` returns `graphify_out_accesses: 0` (static, per D-14)
