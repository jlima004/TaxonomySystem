<action>
Create `src/cli/graph_read_model.ts` with:

1. **Arg parsing:** `parseGraphBuildArgs(argv: string[])` — returns `{ json: boolean; help: boolean; dryRun: boolean; skipGuardrails: boolean }`. Flags: `--json`, `--help`, `--dry-run`, `--skip-guardrails`.

2. **Help text:** `printHelp()` — documents `graph:build` usage, flags, and workflow steps. Mirror `alias_integrity.ts` help style.

3. **Input loading:** `loadGraphInputs(baseDir: string)` — uses `resolveReadablePath` for each of `GRAPH_ALLOWED_PRODUCTION_INPUTS` (3 files), reads and JSON-parses them. Returns `BuildOlfactoryGraphInput`.

4. **Guardrail execution (GVAL-05):** `runGuardrails(srcDir: string)` — spawns 4 npm scripts sequentially via `execFileSync('npm', ['--prefix', srcDir, 'run', scriptName, ...args])`:
   - `typecheck`
   - `test`
   - `alias:integrity -- --json`
   - `verify:integrity -- --json`
   Returns structured `{ passed: boolean; results: { name: string; exitCode: number; output: string }[] }`.

5. **Main workflow:** `runGraphBuildCli(argv: string[])` returning exit code:
   - Parse args → if `--help`, print help, return 0
   - Determine output dir: `--dry-run` → `/tmp/graph-read-model-dry-run/` else `GRAPH_OUTPUT_POLICY.sanctioned_output_path`
   - Resolve base dir (parent of `src/` cwd)
   - Discover protected files + capture pre-digests (skip if `--dry-run`)
   - Load inputs from `GRAPH_ALLOWED_PRODUCTION_INPUTS`
   - Build graph: `buildOlfactoryGraph(input)`
   - Validate graph: `validateOlfactoryGraph(graph)` — if `!result.ok`, print errors, return 1
   - Write graph: `writeGraphOutput(graph, resolvedOutputDir)`
   - Run boundary audit (skip if `--dry-run`): `runBoundaryAudit(outputPath, baseDir, preDigests)`
   - If `!auditResult.ok`: print mutation detected warning, return 1
   - Run GVAL-05 guardrails (skip if `--skip-guardrails` or `--dry-run`)
   - If `--json`: print combined JSON report `{ graph_output, boundary_audit, guardrails }`, return 0
   - Else: print human-readable summary, return 0

6. **Entrypoint guard:** `if (process.argv[1] && import.meta.url === pathToFileURL(process.argv[1]).href)` pattern.

7. **Error handling:** Catch `GraphWriteError` for structured error reporting. General errors: `console.error` + exit 1.
</action>
