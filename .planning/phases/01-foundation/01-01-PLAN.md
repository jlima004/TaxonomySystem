# Plan 01-01: Setup TypeScript, Vitest e Arquitetura Base

## Objective

Inicializar o pacote `src/` como um projeto TypeScript independente (mesma estrutura do engine existente), configurado com strict mode, ESM, Vitest, e zero runtime dependencies — pronto para receber a implementação do taxonomy builder.

## must_haves

1. `src/package.json` existe com `"type": "module"`, scripts `build` e `test`, somente devDependencies
2. `src/tsconfig.json` replica as flags strict do engine existente (`noUncheckedIndexedAccess`, `exactOptionalPropertyTypes`)
3. `src/vitest.config.ts` configurado para rodar testes em `src/tests/`
4. `npm install` e `tsc --noEmit` executam com sucesso no diretório `src/`
5. Zero runtime dependencies — apenas `typescript` e `vitest` como devDeps
6. Estrutura de diretórios modular criada
