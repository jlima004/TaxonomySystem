### Task 1: Criar `normalize_unicode.ts` — NFD decomposition + diacritic removal + ligature expansion

<read_first>

- src/tests/normalization.test.ts
</read_first>

<action>
Criar `src/normalizer/normalize_unicode.ts`:

```
export const normalizeUnicode = (input: string): string => {
  return input
    .normalize('NFD')
    .replace(/[\u0300-\u036f]/g, '')
    .replace(/œ/g, 'oe')
    .replace(/æ/g, 'ae')
    .replace(/Œ/g, 'OE')
    .replace(/Æ/g, 'AE')
}
```

Responsabilidades:
- NFD decomposition (separa base + combining marks)
- Remove combining diacritical marks (U+0300–U+036F)
- Expande ligatures: œ→oe, æ→ae, Œ→OE, Æ→AE

NOTA: Esta função NÃO faz lowercase — isso é responsabilidade de `normalizeCase`.
</action>

- `normalizeUnicode('ÆTHER')` → `'AETHER'`
- `normalizeUnicode('already clean')` → `'already clean'` (idempotente para inputs limpos)
- Não altera case: `normalizeUnicode('ÜBER')` → `'UBER'` (mantém uppercase)
</acceptance_criteria>

### Task 7: Criar `singularize.ts` — plural → singular com dictionary-first + suffix fallback

<read_first>

</read_first>

<action>
Criar `src/normalizer/singularize.ts`:

```
export const singularize = (input: string): string => {
  // Se contém underscores, singularizar cada token individualmente
  if (input.includes('_')) {
    return input
      .split('_')
      .map(singularizeToken)
      .join('_')
  }
  return singularizeToken(input)
}

const singularizeToken = (word: string): string => {
  // Camada 1: dictionary lookup
  const irregular = lookupIrregularPlural(word)
  if (irregular !== undefined) return irregular

  // Camada 2: suffix stripping rules (ordem importa — mais específico primeiro)
  // ies → y (berries → berry)
  if (word.endsWith('ies') && word.length > 3) return word.slice(0, -3) + 'y'
  // es → apenas após s, x, z, ch, sh (classes → class, boxes → box)
  // NÃO strip genérico de es — quebraria roses → ros, spices → spic
  if (/(ses|xes|zes|ches|shes)$/.test(word)) return word.slice(0, -2)
  // s → remove sufixo s (exceto ss para não quebrar moss → mo)
  if (word.endsWith('s') && !word.endsWith('ss') && word.length > 2) return word.slice(0, -1)

  return word
}
```

Regras de suffix stripping (D-05, corrigidas):
- `ies` → `y` (berries → berry, só se length > 3)
- `es` → `` APENAS após s/x/z/ch/sh (regex contextual — evita roses → ros)
- `s` → `` (exceto palavras terminadas em `ss` como moss, grass)

NOTA: A regra genérica `es` foi removida porque é semanticamente perigosa. Palavras como `roses`, `spices`, `notes` terminam em `es` mas o singular é `rose`, `spice`, `note` (basta remover o `s`). O dictionary-first cobre os irregulares perfumísticos (`mosses`, `leaves`), e a regra contextual `es` cobre sibilantes (`classes`, `boxes`). O fallback `s` cobre o resto.
</action>
