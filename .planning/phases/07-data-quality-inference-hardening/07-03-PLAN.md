- support threshold >= 3.
- normalized_support threshold >= 0.05.
- placement_score threshold >= 0.35.
- placement_score = 0.60 * min(1, support / 10) + 0.30 * min(1, normalized_support / 0.20) - 0.10 * noise_penalty.
