# Phase 31 — Rosewood Add Target Planning : Context

## Motivação
A Fase 30 concluiu que `boi_de_rose` (e truncamentos como `boi`) são variantes de `bois_de_rose` / `rosewood` / `pau_rosa`. No entanto, nenhum desses targets canônicos existia no sistema, impedindo a criação imediata de um alias. Esta fase planeja a adição do target correto (`rosewood`) para viabilizar futuras deduplicações.

## Objetivos da Adição de Target
1. Inserir `rosewood` como seed descriptor canônico, já que é o termo padrão da indústria para "pau-rosa".
2. Posicioná-lo na família `woody` e subfamília `woody_dry`.
3. Preparar o terreno para que, em fases subsequentes, `bois_de_rose`, `boi_de_rose`, `boi` e `pau_rosa` possam ser mapeados como aliases para este novo target.

## Base de Conhecimento
- **Fase 30:** Investigação e encerramento (Closure/Research) que apontou a necessidade de criação deste target.
- **Taxonomy v2:** O sistema atual de seeds e aliases.
