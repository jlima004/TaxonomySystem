## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI argv → alias integrity parser | Untrusted flags control output mode and path selection. |
| Filesystem JSON → validator | Seed aliases, compiled taxonomy, and exception policy content may be malformed or drifted. |
| Exception policy data → PASS/FAIL decision | Incorrectly validated exception entries could create a false PASS. |
| Validator result → shell exit code | Output formatting or exit-code mapping could hide a failing live-data proof. |
